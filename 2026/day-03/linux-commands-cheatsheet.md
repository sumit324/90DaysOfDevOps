## Process management.

1) $top = view CPU and MEMORY usage.
2) $htop = Intarctive process view.
3) $ps = view running process.
4) $ps -aux = detail process view (CPU/MEMORY).
5) $kill = stop the process.
6) $kill Pid = kill perticular process.
7) $uptime = check system running time.
8) df -h = check disk space usage.
9) free -h = check memory status.

## File system.

  1) $mkdir = make directory(folder).
  2) $touch = make file.
  3) $rm -r directory_name = remove directory.
  4) $cat = display output in file.
  5) $chmod = change file perrmission.
  6) $cp = copy file.
  7) $mv = move file AND Rename file.
  8) $head = read file top 5 line.
  9) $wc = word count
  10) $tail = last line show.
  11) $less = page by page detail.
  12) $chown = change owner.
